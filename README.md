# 🏥 Healthcare Excel Analytics Dashboard

## 📌 Project Overview

This project is an **Excel-based Healthcare Analytics Dashboard** designed to analyze and visualize key healthcare data.
It helps in tracking performance metrics, identifying trends, and supporting data-driven decision-making.
